Instalación y Uso

Sistema listo para usar en Unity. Solo requiere configuración básica.

1

Descargar Package

Descarga el .unitypackage desde GitHub en el apartado Releases

2

Importar en Unity

Abre Unity 2022.3+ e importa el package completo al proyecto

3

Configurar Modelo

Selecciona tu modelo 3D y activa "Read/Write Enabled" en Import Settings

4

Añadir y Fracturar

Añade el componente "FractureController" y configura la herramienta
